# Subgraphs

## Setup

### Init

```bash
$ graph init --studio <studio-name>
```

### Auth

```bash
$ graph auth --studio <secret>
```

### Deploy

```bash
$ graph codegen && graph build
```

```bash
$ graph deploy --studio <studio-name>
```
